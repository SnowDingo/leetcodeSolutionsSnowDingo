# Cheatsheet

This is a leetcode cheatsheet for Python


## Help Functions
`max()` -> Finds the maximum of a list or a variable. Can use `key=len` to find the longest string in an array as seen in problem 1071.

## Strings
`in` -> Used in if statements to check if x is a substring of y
`*` repeat the string `x` n times.

## Array
`.append()` -> Insert an element to the array
